## **📝 IMPORTANT NOTES**

### **Password Policy**
- All passwords follow the format: `ZMC@[Role][Year]!`
- Minimum 12 characters
- Include uppercase, lowercase, numbers, and special characters
- Passwords must be changed on first login
- Passwords expire every 90 days

### **Security Guidelines**
1. **First Login:** Change default password immediately
2. **Password Security:** Do not share passwords with anyone
3. **Session Management:** Log out when not using the system
4. **Access Control:** Only access areas relevant to your role
5. **Reporting:** Report any suspicious activity immediately

### **Account Management**
- **Account Creation:** Only system administrators can create accounts
- **Role Assignment:** Roles are assigned based on job function
- **Access Review:** Access rights are reviewed quarterly
- **Deactivation:** Accounts are deactivated when staff leave

## **🚀 QUICK LOGIN GUIDE**

### **Step 1: Access the System**
1. Open web browser
2. Navigate to: `https://portal.zmc.co.zw`
3. Click on "Staff Login"

### **Step 2: Enter Credentials**
1. **Username:** Your assigned username
2. **Password:** Your assigned password
3. Click "Login"

### **Step 3: First Login Setup**
1. You will be prompted to change your password
2. Enter current password
3. Create new password (follow security guidelines)
4. Confirm new password
5. Click "Update Password"

### **Step 4: Dashboard Access**
1. You will be redirected to your role-specific dashboard
2. Access features based on your assigned role
3. Navigate through the system using the menu

## **🔄 PASSWORD RESET PROCEDURE**

### **Self-Service Reset**
1. Go to login page
2. Click "Forgot Password"
3. Enter your email address
4. Check email for reset link
5. Create new password
6. Login with new password

### **Admin-Assisted Reset**
1. Contact IT Support
2. Provide identification details
3. Temporary password will be issued
4. Login with temporary password
5. Change to permanent password
